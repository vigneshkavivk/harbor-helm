# Dashboard Overview

The CloudMaSa dashboard is your central hub for managing all DevOps activities.

## Top Navigation Bar

- **Logo + Title**: Click to return to home
- **User Menu**: Profile, settings, logout

## Left Sidebar

- **Workspace**: Manage projects
- **Cloud Connector**: Link AWS, GCP, Azure
- **Clusters**: View and manage Kubernetes clusters
- **Work Flow**: Create and monitor CI/CD pipelines
- **SCM Connector**: Connect GitHub/GitLab repositories
- **Tools**: Access Argo CD, Jenkins, etc.
- **Database**: Manage PostgreSQL, MySQL, etc.
- **Policies**: Set security and compliance rules
- **Masa Bot**: Chat with AI assistant
