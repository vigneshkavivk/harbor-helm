# Team Collaboration

Work together seamlessly with role-based access and shared workspaces.

## Roles

- **Admin**: Full access, manage users
- **Editor**: Create/edit resources
- **Viewer**: Read-only access

## Inviting Team Members

1. Go to **Settings > Team**
2. Click **+ Invite User**
3. Enter email and select role
4. Send invitation

## Shared Workspaces

- All team members see the same workspace
- Changes are synced in real-time
- Audit logs track who did what
